# Mini E-commerce Store Data Analysis Project (SQL) - Day 3

## 📌 About the Project
Building a relational database from scratch for an e-commerce store consisting of three related tables (Products, Customers, Orders) and applying advanced queries to extract statistics.

## 🛠️ Skills Applied Today
- Designing tables and defining relationships and foreign keys.

- Filtering and sorting data (WHERE, BETWEEN, ORDER BY).

- Grouping and calculating statistics (COUNT, SUM, GROUP BY).

- Advanced table joining (INNER JOIN, LEFT JOIN).

- Calculating the store's total sales.

- Programmatically determining the number of products in each category.
